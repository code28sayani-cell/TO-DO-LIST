# 📝 To-Do List App

Hey! 👋
This is a simple yet modern **To-Do List web app** I built using **HTML, CSS, and JavaScript**. The goal was to keep things clean, responsive, and actually pleasant to use — not just functional.

---

## ✨ What this app can do

* Add new tasks easily
* Mark tasks as completed ✔️
* Delete tasks you no longer need 🗑️
* Save everything automatically (thanks to localStorage 💾)
* Works smoothly on both desktop and mobile

---

## 🎨 Design approach

I wanted something that looks modern but not overcomplicated. So I went with:

* A **dark blue theme** (easy on the eyes 🌙)
* A centered **card layout** with soft shadows
* Subtle animations for interactions
* Clean typography using **Inter font**

Overall, the focus was on **good UX + simplicity**.

---

## 🛠️ Built with

* HTML – for structure
* CSS – for styling and animations
* JavaScript – for functionality
* localStorage – to save tasks

No frameworks, just pure basics 🚀
